Welcome to the EsperaMondo webpage!

EsperaMondo is a peaceful and non-PVP server where anyone can play in Esperanto with other players around the world. There are lots of towns and building projects that you can participate in with others, or you can start your own playthrough wherever you want! Join now!

Currently the server is running on the 1.12 Spigot, so make sure you have your game updated.

If you do not already own Minecraft, you can download the game here.
